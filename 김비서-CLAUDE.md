# 김비서 — 업무 대시보드

PT 트레이너/피트니스 사업가를 위한 통합 업무 관리 및 매출 분석 대시보드.

## 주요 기능

- **할 일 관리**: 일일 업무 체크리스트
- **일정 관리**: 월간 스케줄 및 예약 현황
- **매출 분석**: 실시간 매출 데이터 및 트렌드
- **다이어그램**: 업무 흐름도 및 매출 구조 시각화

## 파일 구조

```
├── 업무대시보드.html          # 메인 대시보드 (할 일, 일정, 요약)
├── 매출보고서.html             # 매출 분석 및 차트
├── 마케팅업무흐름.svg          # 업무 프로세스 다이어그램
├── 크몽판매전략대시보드.html    # 크몽 채널 분석
└── 아임웹_자사몰_버전.html      # 아임웹 자사몰 통합
```

## 사용 기술

- **HTML5**: 마크업 및 페이지 구조
- **CSS3**: 반응형 디자인, 그리드/플렉스박스
- **Vanilla JavaScript**: 데이터 처리 및 차트 렌더링
- **Chart.js / Apexcharts**: 데이터 시각화
